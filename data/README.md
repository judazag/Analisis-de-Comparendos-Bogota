# Carpeta de Datos

Debido al tamaño del dataset (402,500 registros), el archivo original no se encuentra en este repositorio.

### Instrucciones para obtener los datos:
1. Visite el [Portal de Datos Abiertos de Movilidad](https://datos.movilidadbogota.gov.co/datasets/movilidadbogota::comparendos-2019-bogota-d-c/about).
2. Descargue el archivo en formato **CSV**.

*Nota: El notebook está configurado para leer el archivo con este nombre y estructura de 18 columnas.*
